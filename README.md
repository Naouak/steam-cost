# Steam Cost

An user script to get some stats from your Steam payments page ( https://store.steampowered.com/account/ ).

## How to install on Chrome

- Download the javascript file "steam-cost.user.js"
- Open the extension tab in your chrome (menu > tools > extensions)
- Drag n drop the javascript file in the extension tab.
- Open https://store.steampowered.com/account/ and enjoy
